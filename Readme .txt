Readme файл модуля Б

интерфейс созданнй с помощью Qt Designer работает следующим образом
On - включение работы манипулятора
Pause - пауза - приостановка  работы манипулятора с возможностью дальнейшего продолжения при повторном нажатии
Stop - остановка работы манипулятора
Actual tool Pose - актуальная  позиция
Manual Cart - переключить на ручной режим
Manual Joint - переключить на авто режим
Видео  с камер - изоображения передающие камеры манипулятора
For Manual - ползунки для ручного управления моторами
stopped - остановить
work -  начать работу
off - выключить
wait - ждать
gripper On - включить гриппер
Стиль  движения - выбрать стиль движения(возможно только один)
Цикл - задать цикл работы манипулятора
Таблица состояния моторов - демонстрирует состояние моторов, показывает тики, радианы, градусы, температуру каждого из  моторов
Таблица для  назначения координат - вписываем  координаты для дальнейшего перемещения манипулятора в данный точки
Вывод  Логов - выведение действия манипулятора в текущий момент
Цвета объекта - подсвечивается цвет - определяемый зрением
Форма объекта - подсвечивается объект - определенный зранием
Считывание штрих-кодов - возможность считывания штрих-кодов с объектов

Индикаторы загараются при виде необходимого цвета

Индикаторы загараются при нужном моменте работы робота
